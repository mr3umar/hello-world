Hello World
=========

A silly library that return a hello-world message

## Installation

  `npm install @mr3umar/hello-world`

## Usage

    var greeter = require('@mr3umar/hello-world');

    var msg = greeter();
  
  
  Output should be `Hello World`


## Tests

  `npm test`
